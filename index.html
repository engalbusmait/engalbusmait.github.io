Featured Projects

01 / AI System
AI-Powered Bilingual Forensic Reporting System
AI / LLM On-Premises Forensics

An on-premises AI system using locally hosted Ollama Qwen2 7.7B LLM to automatically generate bilingual Arabic–English forensic investigation reports for court submission — reducing preparation time by 80%.

Designed and developed for the Directorate of Anti-Cyber Crimes, this system runs entirely on the directorate's internal infrastructure with no external API dependency, ensuring full data confidentiality and compliance with law enforcement security requirements.

The system automatically produces bilingual (Arabic–English) investigation reports formatted for court submission, incorporating a custom forensic terminology dictionary maintaining accuracy across critical legal terms such as chain of custody, evidence classification, and cybercrime statutes.

By keeping the AI model within the secure environment, sensitive case data never leaves the network. The solution standardized documentation quality across investigators and enabled non-English-speaking staff to produce court-admissible bilingual reports independently.

02 / Web System
Staff Leave & Night Shift Management System
Web App HR Automation Internal Tool

Internal web-based system to automate and centralize annual leave requests, approval workflows, and night shift scheduling for directorate personnel — replacing all manual paper-based processes.

Developed to streamline HR operations within the directorate, this system handles submission, approval, and tracking workflows for leave requests, maintains accurate leave balances per employee, and manages night shift assignments and compensatory leave entitlements.

Features include: real-time staff availability overview for supervisors, automated report generation for HR and administrative use, and a full audit trail of all leave and shift records. The solution significantly reduced administrative overhead and improved scheduling transparency across the directorate.

03 / Case Management
Child Protection Case Management System
Full-Stack Air-Gapped React + Flask

Comprehensive offline case management platform handling all aspects of cybercrime cases involving minors — built on Python Flask + React with full Arabic RTL interface, running on an air-gapped local network.

Architected for 20–40 concurrent users on a local network, this platform includes: case management with party tracking and procedural workflows, digital evidence (seized devices) inventory with chain-of-custody records, prosecution orders tracking, and events/training scheduling with smart pre-event notifications.

Features a multi-period statistical dashboard (monthly, quarterly, semi-annual, annual), role-based access control (Superadmin, Admin, User, Read-only), immutable audit logging, dynamic custom field configuration through the admin panel, secure document attachment management organized by case, and automated daily/weekly database backups.

Designed to operate in a fully air-gapped environment with bilingual data entry support and full Arabic RTL interface compliance.
